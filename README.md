# Mmph! — The Wimpy Kid Word Game

A browser-based word party game inspired by **Diary of a Wimpy Kid: Hot Mess** by Jeff Kinney.

> "Describe the secret word without saying any forbidden words — or Rodrick hits the buzzer on you!"

## How to Play

Open `mmph-game.html` in any modern browser — no installation or internet required.

### Versus Mode (2 teams)
1. Enter team names and choose game settings
2. The clue-giver looks at the secret word and describes it to their team
3. **Cannot say** any of the 5 forbidden words shown on the card
4. Opponents press the **BUZZER** if a forbidden word is said (−1 point)
5. Score as many correct guesses as possible before the timer runs out
6. Teams take turns until all rounds are complete — highest score wins!

### Solo Challenge Mode (1 player)
1. Select **Solo challenge** on the setup screen
2. Race against the clock to describe as many words as possible
3. Your score is saved to a local leaderboard — try to beat your best!

## Features

- **Two game modes** — Versus (2 teams) and Solo challenge (1 player)
- **Timed turns** — 30 / 45 / 60 / 90 / 120 seconds per turn
- **4 word categories** — Animals, Food, School, Everyday life (or all mixed)
- **Sound effects** — synthesized via Web Audio API (no external files)
- **Local leaderboard** — top scores saved in your browser for Solo mode
- **New Record badge** — celebrates when you beat your personal best
- Fully offline, single HTML file, mobile-friendly

## Inspiration

This game is based on the **Mmph!** word game featured in *Diary of a Wimpy Kid: Hot Mess* by **Jeff Kinney**. All game concept credit goes to Jeff Kinney and Wimpy Kid, Inc. This is an unofficial fan-made digital adaptation for personal/educational use.
